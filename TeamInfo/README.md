## Project Overview
This project explores the factors influencing salaries in the Artificial Intelligence sector. Using a dataset of 2,000 job listings, we perform:
1. **Exploratory Data Analysis (EDA):** Visualizing salary distributions and industry hiring patterns.
2. **Statistical Analysis:** Using **ANOVA** to test the impact of experience levels on salary.
3. **Machine Learning:** Implementing **Linear Regression** to provide real-time salary estimates.
4. **Deployment:** A **Streamlit** web application for interactive user predictions.
